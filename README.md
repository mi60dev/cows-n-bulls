# Cows & Bulls Game 

written in Laravel & Vue js

## Installation

### Step 1
```
composer install
```
### Step 2
```
npm i
```
### Step 3
```
npm run dev
```
### Step 4
```
php artisan key:generate
```
### Step 5
```
php artisan migrate
```
### Step 6
```
cp .env.example .env
--> Setup DB
--> Set APP_URL in .env (used for the sanctum auth)
```
